基期、现期

如2017年比2016年增加300元， 2017年为现期，2016年为基期。

增长量

增长量 = 现期量 - 基期量

增长率

增长率 = 增长量 / 基期量 * 100%

如2017年100w，2018年130w，增长率为 （130 - 100） / 100

速算方式： 增长量 / 基期

又名： 比较数、增长率、增长幅度、增长速度、增长势

例题1

2017产值144w，比2016增长20%，求2016产值与2017增长量

1. 2016产值 = 144 / (1 + 0.2) = 120w

2. 2017增长量 = 2017产值 - 2016产值 = 24w

例题2

2017比2016增长3.2倍，2017是2016的多少倍

（1 + 3.2 ）：1 = 4.2倍

间隔增长率

出现题型： 间隔一年

如2018到2019增长率为r1, 2019到2020增长率为r2， 则2018到2020增长率为r

公式： r = r1 + r2 + r1 * r2

混合增长率

大小居中，偏量大一方

示例一
地产贷款余额1.04w亿，同比增长21.4%，房产贷款余额3.2w亿，同比增长12.3%。
求房地产贷款余额同比增速约为？x
A. 14.4% B. 12.3% C. 19.3% D. 24.4%
解：12.3% < ? < 21.4%
因为3.2w > 1.04，往12.3%偏移，所以答案为A。

示例二
全国水品产量2700w吨，同比增长3.2%，其中养殖水产品产量2114w吨，同比增长4.13%
求非养殖水产品同比增长？
A. -20% B. 0% C. 5% D. 10%
解：
2114w  n   2700w  m   得出非养殖 600w
4.1% <---> 3.2% <---> ?
     0.9%         x
量跟距离成反比 即 n : m = 600 : 2114 = 1 : 3.5
等价于公式 n/m = 1/3.5
得出 m = 3.5n
即 x = 0.9% * 3.5 ~= 3.2%, 所以答案为B。

年均增长量

年均增长量 = （现期量 - 基期量）/ 年数

年均增长率

现期量 = 基期量*(1+年均增长率)^年数

1 + n*r = 现期/基期

例题3

2015营收100w，营收年均增长率为10%，2018年营收为多少

100w * (1+0.1)^3 = 133.1w

例题4

2015营收100w，2018年营收133.1w, 2015-2018年均增长率为多少

100w * (1+r)^3 = 133.1w => 开立方

增长率比较

现期a, 增长率r，a大r大则大，一大一小看a*r, 正大于负

平均值增长率

如2018年北京房地产销售额增速为a，销售面积增速为b，均价增速为？

公式： 均价增速 = (a-b)/(1+b)

基期比重/倍数

如2017年，产量A, 增长率r1, 产量B, 增长率r2, 求2016年A占B约多少

x = (A/(1+r1))/(B/(1+r2))

公式： A/B * (1+r2)/(1+r1)

两期比重比较大小

如2017年，全国增长率r1, 某省增长率r2, 求2016年某省占全国比2017年上升还是下降？

r1 > r2 下降

r1 < r2 上升

两期比重差

如2017年，某省产量B, 全国增长率r2, 产量A, 增长率r1, 求2016年AB基期的比重差x

差 = A/B * (r1-r2)/(1+r1)

A/B 部分/整体 < 1

A/B/(1+r1) < 1

步骤：

1. 先判断上升还是下降

2. r1 - r2 得出差

3. 再乘以 A/B/(1+r1) 估算

同比

如 2019.9 ：2018.9

环比

现在统计周期与上一个统计周期相比较

如 2019.9-12月 ：2019.4-8月

比重

如2019年全国gdp为100w亿, 第一产业为20w亿, 占全国比重为20 ：100 = 20%

顺差

出口-进口

逆差

进口-出口

GDP=增加值

恩格尔系数

食品支出总额占个人消费支出总额得比重

比重越低，反应生活水平越高

基尼系数

衡量一个国家或地区居民收入差距得常用指标

介于0-1之间，越接近0，代表收入差距越小，越接近1，代表收入差距越大。

截位直除法

1. 估算原则： 保留3位有效数字

2. 判断原则： 选项首位不同，保留2位；选项首位相同，保留3位

3. 一步除式： 只估算分母

如： 12345/1639 ~= 12345/1670

如： 12345/1685 ~= 12345/1690 ~= 12345/1700

4. 多步连除：分子分母同时截位

例题1

图书总藏量79092万册，比上年末增长5.6%，电子图书50674万册，比上年末增长34.2%，求电子图书增长册数是总藏量的多少倍。

((50674/(1 + 0.342)) * 0.342)
/
((79092/(1+0.056)) * 0.056)
=>
50674/79092 * 1.056/1.342 * 0.342/0.056
=> 5/8 * 1.06/1.34 * 6
=> 30/8 * 1.06/1.34
=> 4- * 1.06/1.34
~= 3

~= 3.4

例题3

出口额26.5亿，同比增长24.9%，求增长多少亿

(26.5/(1 + 0.24)) * 0.24
~= (26.5/(5/4)) * (1/4)
=> 26.5 / 5 = 5.3

增长率与现期求增长量

设增长率r = 1/n

公式=> 增长量=现期量/(n+1)

如例3

增长率24.9 ~= 1/4
n 为4
公式 26.5/(4+1)
=> 26.5 / 5 = 5.3

分数比较

1. 首位直除

如 13/78 比 27/124， 13/78 ~= 0.1+，27/124 ~= 0.2+， 即13/78大

2. 变化速度

如 13/78 比 27/151， 分子13 * 2+ = 27，分母78 * 2- = 151， 即13/78 * 2+/2- = 27/151，2+/2- = 1+，说明13/28需要1倍多才等于27/151
