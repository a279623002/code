# GPU-Mall 算力调度平台

## 项目概述
基于电价策略的AI训练任务调度平台，通过智能任务编排与算力调度，实现算力资源的合理利用与成本优化。

## 技术栈
- 后端框架：Go
- 监控系统：Prometheus
- 调度引擎：自研任务编排系统
- 数据库：MySQL/Redis

## 核心功能
1. **电价感知调度**：根据实时电价动态调整训练任务优先级与执行时间
2. **任务编排管理**：支持多任务队列、依赖关系配置与智能调度策略
3. **算力资源管理**：收录登记自有算力与外部算力资源，统一资源池管理
4. **训练工况监控**：实时采集与记录任务运行状态、GPU利用率等关键指标
5. **外部算力对接**：支持对接第三方算力平台，实现算力弹性扩展

## 主要负责工作
1. **Prometheus监控集成**：
   - 开发Prometheus查询模块，实时获取GPU训练工况数据
   - 设计任务监控指标体系，记录训练进度、资源使用、异常状态
   - 实现监控数据持久化存储与历史查询功能

2. **任务编排系统开发**：
   - 设计任务队列与调度算法，支持优先级、依赖关系、电价策略调度
   - 实现任务生命周期管理，从提交、排队、执行到完成的全流程控制
   - 开发任务调度引擎，支持动态调整执行计划

3. **算力资源管理**：
   - 开发算力资源收录登记系统，支持GPU、服务器等资源信息管理
   - 实现资源池调度，根据任务需求智能匹配最优算力资源
   - 开发资源利用率统计与分析功能

4. **外部算力对接**：
   - 设计并实现第三方算力平台对接接口
   - 开发跨平台任务分发与结果同步机制
   - 实现算力成本核算与收益分析功能

## 项目亮点
- 通过电价策略调度，有效降低训练成本，提升资源利用效率
- 统一管理自有与外部算力资源，实现弹性扩展
- 完整的任务监控与调度体系，保障训练任务稳定运行
