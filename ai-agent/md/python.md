# Python面试八股文

## 基础语法

### 1. Python的特点
- 解释型、面向对象、动态类型
- 语法简洁，可读性强
- 丰富的第三方库
- 跨平台
- 胶水语言，可与其他语言集成

### 2. Python2和Python3的主要区别
- print从语句变为函数
- 字符串编码：Python3默认Unicode
- 整数除法：3/2在Python3返回1.5
- xrange变为range
- 异常捕获语法变化
- 元类和装饰器语法优化

### 3. 列表和元组的区别
- 列表是可变的，元组不可变
- 列表用[]，元组用()
- 元组可作为字典的key，列表不行
- 元组性能稍好，内存占用更小

### 4. 字典的底层实现
- Python3.7+字典是有序的
- 底层使用哈希表
- 哈希冲突使用开放寻址法
- 动态扩容机制

### 5. 装饰器的作用和实现
- 在不修改原函数代码的情况下扩展功能
- 使用@语法糖
- 底层基于闭包实现
- 常见用途：日志、计时、权限验证、缓存

## 进阶特性

### 6. 生成器和迭代器的区别
- 迭代器实现了__iter__和__next__方法
- 生成器是特殊的迭代器，使用yield关键字
- 生成器表达式：(x for x in range(10))
- 节省内存，惰性计算

### 7. *args和**kwargs的用法
- *args：接收任意数量的位置参数，返回元组
- **kwargs：接收任意数量的关键字参数，返回字典
- 函数定义时用于接收不定参数
- 函数调用时用于解包参数

### 8. 闭包的理解
- 内部函数引用了外部函数的变量
- 外部函数返回内部函数
- 外部函数的变量不会被回收
- 常用于工厂函数、装饰器

### 9. 垃圾回收机制
- 引用计数为主
- 标记-清除和分代收集为辅
- 循环引用需要手动处理或使用weakref
- gc模块可手动控制

### 10. 深拷贝和浅拷贝的区别
- 浅拷贝：只拷贝对象本身，不拷贝内部元素
- 深拷贝：递归拷贝对象及其内部元素
- 使用copy模块的copy()和deepcopy()

## FastAPI与Web开发

### 11. FastAPI的特点
- 高性能（基于Starlette和Pydantic）
- 自动生成API文档（Swagger/ReDoc）
- 异步支持
- 类型提示，数据验证
- 依赖注入系统

### 12. Pydantic的作用
- 数据验证和设置管理
- 基于类型提示
- 自动类型转换
- 序列化和反序列化
- 自定义验证器

### 13. 异步编程在Python中的实现
- async/await语法
- asyncio事件循环
- asyncio.create_task()创建任务
- asyncio.gather()并发执行多个协程
- aiohttp异步HTTP客户端/服务器

### 14. Python中的并发和并行
- 多线程：由于GIL限制，适合IO密集型
- 多进程：可利用多核CPU，适合CPU密集型
- 协程：单线程内切换，开销最小，适合IO密集型
- concurrent.futures模块提供高级接口

## 项目经验相关

### 15. LangChain的核心概念
- Chain：多个组件串联执行
- LLM：大语言模型抽象
- PromptTemplate：提示词模板
- Memory：对话记忆
- Agent：智能体，可使用工具
- Tools：工具调用
- Vector DB：向量数据库

### 16. FAISS向量数据库的使用
- 创建索引（IndexFlatL2、IndexIVFFlat等）
- 向量添加和检索
- 相似度计算（L2距离、内积）
- 索引的保存和加载
- 批量操作优化性能

### 17. 大文件处理方案
- 流式读取，避免一次性加载到内存
- 分批处理，batch_size控制
- 临时文件存储中间结果
- 使用生成器逐行处理
- 内存映射（mmap）

### 18. Unstructured文档解析
- 支持多种格式：PDF、Word、PPT、Markdown等
- 自定义Loader扩展
- 文档分块策略
- 元素类型区分（标题、段落、表格等）
- 中文文档优化处理

### 19. 模型预加载和缓存机制
- 服务启动时预加载模型到内存
- 全局单例模式
- 向量库缓存避免频繁磁盘IO
- LRU缓存策略
- 懒加载可选配置

### 20. 日志管理最佳实践
- 使用Loguru等结构化日志库
- 分级日志（DEBUG、INFO、WARNING、ERROR）
- 按时间/大小轮转日志文件
- 不同模块独立日志
- 包含关键信息：时间、级别、模块、消息
